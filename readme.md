# Color Scheme Generator

Generate color schemes for your web page.

This is a simple, front-end web application that allows you to generate harmonious color palettes. It fetches color data to provide you with schemes you can preview and use in your own projects.

## About

The Color Scheme Generator provides a user-friendly interface to create color combinations. Whether you need a monochromatic palette for a minimalist design or a complementary scheme for a vibrant look, this tool helps you explore and select colors efficiently.

## Features

*   **Generate Schemes:** Fetches color schemes dynamically using JavaScript.
*   **Preview Colors:** Visually see your generated color palette.
*   **Clean Interface:** Simple and focused layout built with HTML and CSS.

## Technologies Used

*   **HTML:** Structures the web page.
*   **CSS:** Styles the application and ensures a responsive layout.
*   **JavaScript:** Handles the core logic to fetch and display color schemes from an external API.

## Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/PYticPH/Color-Scheme-Generator.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Color-Scheme-Generator
    ```
3.  **Open the `index.html` file** in your preferred web browser.
